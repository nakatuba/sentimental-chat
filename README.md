# Sentimental Chat
## Setup
### frontend
```
docker-compose run --rm frontend npm install
```
### backend
backend/.env
```
SECRET_KEY=xxxx
DEBUG=True
```

## Run server
```
docker-compose up -d
```
* frontend
  * [localhost:3000](http://localhost:3000)
* backend
  * [localhost:8000](http://localhost:8000)
